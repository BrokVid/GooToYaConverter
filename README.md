## Google → Yandex Coords Pro

Инструмент для переноса объектов с Google Maps на Яндекс.Карты (сервис «Народная карта»). Делает конвертацию координат WGS84 → Яндекс, поддерживает автоконвертацию буфера обмена и калибровку под конкретный город.

### Как работает «магия» конвертации
- **Цель:** быстро перенести объекты/точки из Google в Яндекс, когда из-за смещения слои не совпадают.
- **Алгоритм:** базовое преобразование + поправка по опорным точкам (IDW — inverse distance weighting).
- **Калибровка:** чем больше опорных точек по городу, тем точнее будет автоконвертация. Опорные точки хранятся в `calibration.json` рядом с приложением.

### Рекомендованный порядок работы
1) **Ручная проверка смещения.** Введите координату из Google в блоке «Ручная конвертация» и посмотрите, насколько смещён результат в Яндекс.  
2) **Если смещение велико — калибровка:**
   - Включите «Калибровка» (кнопка «Старт калибровки»).
   - Сначала скопируйте координату чётко видимой точки в Google (углы бордюров, разметка, углы футбольных полей, газоны — объекты с нулевой высотой).
   - Затем скопируйте координату **той же точки** в Яндекс.Картах.
   - Повторите для нескольких точек, лучше диагонально разделяющих город.
3) После калибровки включите автоконвертацию и проверяйте новые точки: приложение будет автоматически подхватывать координаты из буфера обмена, конвертировать и возвращать их в буфер.

### Структура проекта (web only)
- `app.py` — Flask-приложение.
- `templates/`, `static/` — фронтенд (HTML/CSS/JS).
- `calibration.json` — опорные точки (создаётся автоматически, если нет).
- `requirements.txt` — зависимости (Flask, pyperclip).
- `run_app.bat` — быстрый запуск в локальном venv.
- `build_exe.bat` — сборка Windows exe (без консоли) через PyInstaller.
- `dist/`, `build/` — артефакты сборки (не коммитятся).

### Запуск локально
```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
python app.py
```
Откроется компактное окно в Edge/Chrome (режим app). Можно также запустить через `run_app.bat`.

### Сборка exe (без консоли)
```bash
build_exe.bat
```
Готовый файл: `dist/GoogleToYandexWeb/GoogleToYandexWeb.exe`.

### Что не коммитится
- Виртуальные окружения (`.venv/`, `venv/`)
- Папки сборки/дистрибутивов (`build/`, `dist/`)
- Временные файлы Python/IDE (см. `.gitignore`)
